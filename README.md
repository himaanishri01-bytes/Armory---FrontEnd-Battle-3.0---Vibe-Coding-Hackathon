# ARMORY — Enterprise Asset Telemetry & Mesh Orchestration

A high-performance, low-latency telemetry dashboard built for absolute data scale visualization. **ARMORY** features a sleek, production-ready glassmorphic UI alongside a responsive real-time data monitoring suite.

## Live Deployment
*  https://armory-front-end-battle-3-0-vibe-co-iota.vercel.app/


## Key Architectural Components

### 1. High-Density Particle Simulation
* **Immersive Visual Engine:** Features an optimized custom HTML5 Canvas particle physics pipeline that spawns highly visible, dark floating Bitcoin symbols (₿) and ambient liquid bubble elements instantly onto the screen to deliver an immediate premium first impression.
* **Clean Identity Gateway:** A streamlined login terminal completely free of version text noise to maintain high-end production aesthetics.

### 2. Core 3-Box Telemetry Grid
* **Visual Topology Map:** A live cluster module capturing standalone node metrics (Node 01: `11ms` and Node 02: `10ms`) alongside structural inline dynamic stream vectors and an integrated network health badge.
* **Interactive Timeline Graph:** A real-time data throughput canvas utilizing full sub-pixel coordinate mapping. Features an asset timeline scrubber tooltip that tracks live performance parameters on crosshair hover.
* **AI Auditing Component:** An automated diagnostic window that evaluates runtime integrity parameters and generates adaptive operational trend logs.

### 3. Simulation & Event Management
* **Jitter Injection Suite:** Interactive debug controls designed to inject synthetic network latency spikes and test system resilience under cluster load.
* **Persistent Event Audit Ledger:** A live, scrollable terminal module logging real-time user handshakes, configuration alerts, and diagnostic changes.

### 4. Multi-Tier Resource Pricing Matrix
* **Product Tiers:** Features three distinct, fully structured product upgrade levels: *Core Pipeline*, *Advanced Mesh*, and *Hyper Mesh*.
* **Localization Switches:** Includes integrated currency localization switches (INR ₹ / USD $) alongside an interactive billing cycle toggle.


## Tech Stack
* **Frontend:** Semantic HTML5 Structure
* **Styling Engine:** Tailwind CSS Engine (Utility-first execution)
* **Interactive Control Logic:** Pure Vanilla JavaScript ES6+ Pipeline & Canvas 2D APIs
* **Typography:** Google Inter & JetBrains Mono Fonts
